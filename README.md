# Simple TODOs App

## About "TODO-aholic"

My first application with MEAN stack.
Here you can create and manage yours ToDos list and be sure, that nobody can access them.

## Help

To run this app:
* **Install NodeJS to yours local machine**
* **Install MongoDB**
* **Run app with "node server.js"**
* **Open your browser**
